## Comments (Giscus)

This site uses Quartoâ€™s built-in **Giscus** integration (GitHub Discussions-backed comments).

### One-time GitHub setup

1. In the GitHub repo `spinkney/spinkney.github.io`, enable **Discussions**.
2. Install the **giscus** GitHub App for the repo: https://github.com/apps/giscus
3. In **Discussions**, create (or choose) a category for comments (e.g. `Announcements` or `General`).

### Config

The site configuration is in `_quarto.yml` under `website.comments.giscus`.
